# FragmentTabApp

An Android application demonstrating Fragments and Material Design, built with Java and XML.


---------

## 🚀 Features

✅ Multi-tab layout using Fragments  
✅ Swipe navigation with ViewPager2  
✅ Material TabLayout integration  
✅ Simple, clean UI
